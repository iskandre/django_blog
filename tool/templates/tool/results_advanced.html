Basic analysis on the users followed by bot and their reactions

This is an example of Bayesian analysis on this account natalie_sn0w

Every graph corresponds to the coefficient in the following model:
p[i])<- b1*follows[i] + b2*followed_by[i] + b3*posts_count[i] + ranef[homeCountry[i]] +

b4*avgLikesPerPost[i] + b5*avgPostingFreq[i] + b6*countriesCount[i] + b7*hashtagsRatio[i] +

b8*bioFlag_indicator[i] + b9*bioLen[i] + b10*bioContacts[i]

Every graph represents the distribution of the coefficient. If the mean value is positive, then it means then the factor positevely affect the probability of following back and other way around.
